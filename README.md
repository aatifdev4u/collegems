# collegems
Sample college management system using codeigniter



## Table Structure:
###### tbl_students
```
->  id
->  studentname
->  college_id
->  gender
->  email
->  course
```
###### tbl_college
```
->	college_id
-> collegename
->  branch

```
###### tbl_roles
```
->	role_id
->  rolename

```
###### tbl_users
```
->	user_id
->  username
->	college_id
->  email
->  gender
->	role_id
->	password
->	confpwd
```
